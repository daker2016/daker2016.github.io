Hyper-V

Hyper-Vとは

Hyper-VはMicrosoft社が提供する仮想化ソフトで、「ハイパーバイザー（Hypervisor）」と呼ばれる仮想化技術がベースとなっています。1台の物理的なハードウェアを仮想化で複数のパーティションに分割し、それぞれOSを動作させることができます。

なぜHyper-Vは必要

  • DockerはLinux OS環境しか動けない-
  • WinやMacで動いているのは、Linuxの仮想環境の上にLinux OSをインストールしてその上でDockerを動かしているケースや、Desktop Docker for Windows や　Desktop Docker for MacのようなDockerの実行に必要なLinuxを含むパッケージをインストールしている。
  • Hyper-V上にLinux仮想マシンを立ててそこでDockerを実行する

  • 入れない場合、下記の処理に入ります。入れましたら、下の「wsl2」に進んでください。
